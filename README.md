# Nodo Site
Landing page for Nodo, an Auckland-based digital systems company.
Created by Pato Anabalón.

## Stack

- Next.js 16 LTS
- TypeScript
- Tailwind CSS
- GSAP with `@gsap/react`
- Atomic design component structure

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Routes

- `/`
- `/services`
- `/about`
- `/case-studies`
- `/contact`
